Sonar app

Sonar is a festival assistant platform, that includes features for programme management, maps and navigation, and a friend tracking system. Its purpose is to keep the user organised, and up-to-date with everything happening at a large scale event.

Wakey wakey app

This is a case study of a wake-up alarm that learns patterns to adapt to its user's habits. The aim was to find a potential solution to a common problem while creating a simple but effective product.

Attenborough

Creative UI project leveraging skrollr.js for a parallax scroll animation effect. The illustrations of Sir David Attenborough come to life when viewed on desktop only.

Disappearing Ones

Disappearing Ones is a point-and-click game focusing on endangered species, with the aim to educate, and raise funds for conservation efforts. All illustrations are my own. I've used HTML, SASS, JavaScript, jQuery and Google maps API for this project.
